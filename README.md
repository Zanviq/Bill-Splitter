# N빵 계산기 (Dutch Pay Master)

본 애플리케이션은 비용 정산 및 개인별 영수증 이미지 생성을 위해 사용됩니다.

## 사용 방법

1. **초기 설정**
   참여 인원 수와 정산 받을 계좌 정보를 입력한 후 시작합니다.

2. **항목 추가**
   지출 항목의 이름과 금액을 입력하고, 비용을 분담할 인원을 선택하여 목록에 추가합니다.

3. **정보 수정**
   상단 인원 목록에서 각 참여자의 이름을 변경할 수 있습니다.

4. **이미지 저장**
   - **개별 저장**: 각 영수증 카드에 위치한 다운로드 버튼을 사용하여 해당 인원의 내역만 저장합니다.
   - **전체 저장**: 상단 메뉴의 전체 저장 버튼을 사용하여 모든 인원의 영수증을 하나의 이미지로 저장합니다.

5. **초기화**
   상단 뒤로가기 버튼을 누르면 입력된 모든 데이터가 삭제되고 초기 화면으로 돌아갑니다.

## 설치 및 실행

1. **저장소 복제**
   터미널에서 다음 명령어를 입력하여 프로젝트를 복제합니다.
   ```bash
   git clone [저장소 URL]
   cd [프로젝트 폴더명]
   ```

2. **의존성 설치**
   npm을 사용하여 필요한 패키지를 설치합니다.
   ```bash
   npm install
   ```

3. **애플리케이션 실행**
   개발 서버를 실행합니다.
   ```bash
   npm run dev
   ```
   실행 후 브라우저에서 `http://localhost:3000`으로 접속합니다.
